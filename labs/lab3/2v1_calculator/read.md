var <идентификатор1> - Объявляет переменную типа double

let - объявляет и присваивает значение
let <идентификатор1> = <число с плавающей запятой> либо
let <идентификатор1> = <идентификатор2>

fn <идентификатор1> = <идентификатор2> либо
fn <идентификатор1> = <идентификатор2><операция><идентификатор3>

print <идентификатор> - выводит значение идентификатора

printvars - Выводит в stdout имена и значения всех ранее объявленных переменных, отсортированных по алфавиту, по одному в каждой строке в следующем формате:
<идентификатор>:<значение> – без каких-либо пробелов и табуляций

printfns - имена и значения всех ранее объявленных функций, отсортированных по алфавиту, по одному в каждой строке в следующем формате:
<идентификатор>:<значение> – без каких-либо пробелов и табуляций

Значение выводится с точностью в 2 знака после запятой

var, let, fn, print, printvars, printfns
